## Overview
This project is an AI-powered maize disease detection system developed using Python and Deep Learning techniques. The system helps identify common diseases affecting maize crops from uploaded images.

## Features
- Upload maize leaf images
- Detect maize diseases automatically
- Predict disease category using trained AI model
- User-friendly interface
- Fast disease prediction

## Technologies Used
- Python
- Jupyter Notebook
- TensorFlow / Keras
- OpenCV
- NumPy
- Flask

## Installation

1. Clone the repository
git clone https://github.com/marufugideon/Maize-Disease-Detection.git

2. Open the project folder
cd Maize-Disease-Detection

3. Create virtual environment
python -m venv venv

4. Activate virtual environment
### Windows
venv\Scripts\activate

5. Install dependencies
pip install -r requirements.txt

6. Run the application
python app.py
